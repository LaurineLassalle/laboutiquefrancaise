# Getting Started
Run the project:
### `symfony serve`
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts
Runs PHPUnit tests:
### `php bin/phpunit`

Create tests:
### `php bin/console make:test`


## Installed bundles
Install PHPUnit :
### `composer require --dev symfony/test-pack`
